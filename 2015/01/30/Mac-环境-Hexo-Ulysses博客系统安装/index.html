Mac 环境 Hexo+Ulysses博客系统安装

hexo简介

hexo是一个基于Node.js的静态博客程序，可以方便的生成静态网页托管在github和gitcafe，作者是来自台湾的tommy351。根据官网介绍，Hexo是一个简单、快速、强大的博客发布工具，支持Markdown格式。

Hexo安装

首先确保Git、Node都安装成功 

npm install hexo -g  #-g表示全局安装,

Hexo创建博客

hexo init <folder>  #执行init命令初始化,例如: hexo init blog
hexo generate       #自动根据当前目录下文件,生成静态网页
hexo server         #运行本地服务

创建博文

hexo new “Mac 环境 Hexo+Ulysses博客系统” #新建博文

[info] File created at /Users/wj/code/blog/source/_posts/Mac-环境-Hexo-Ulysses博客系统.md`

安装问题

  • 安装hexo，如果遇到权限问题用sudo执行。
  • hexo version 2.8.3bug,CSS无法编译,执行 npm install hexo-renderer-stylus@0.1 —save

Ulysses编辑博文

Markdown支持
多文档支持
iCloud支持
多种格式输出

部署github

config.yml添加如下内容：

deploy:
  type: github
  repository: https://github.com/wangjun0511/  wangjun0511.github.io.git
  branch: master

执行部署命令：

hexo d        #部署github

访问 http://localhost:4000 查看效果。
